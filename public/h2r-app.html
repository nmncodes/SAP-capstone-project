Personnel Requisition / Vacancy

The starting point of the recruitment cycle. An internal document used to inform HR of the need to hire an employee for a specific position.

Key Information

T-Codes: PB10 (Create Vacancy) | PBA0 (Process Vacancy)

Requisition Types

  • Internal Requisition - Transfer/Promotion
  • External Requisition - New hire from outside
  • Job-Specific Requisition - Linked to a position
  • General Requisition - Not yet position-specific

Key Fields

Field Description
Position/Job ID ID of vacant position
Organizational Unit Department/section
Required Date Needed by this date
Reason New/Replacement/Project
Qualifications Required skills & experience
Salary Range Budget information

Recruitment & Applicant Selection

Process of issuing Request for Application (RFA) to potential candidates and evaluating their qualifications.

Key T-Codes

HR41: Create RFA
HR42: Change RFA
HR47: Enter Application
HR49: Skill Comparison
HR61: Candidate Evaluation

Evaluation Criteria

  • Experience - Competitiveness vs requirements
  • Skills - Competency assessment
  • Availability - Start date flexibility
  • Fit - Cultural alignment

RFA Process Steps

T-Code Action Description
HR41 Create RFA Enter role, skills, deadline
HR42 Change RFA Modify details
HR43 Display RFA View issued RFAs
HR47 Enter Application Record candidate qualifications
HR49 Skill Comparison Compare applications

Hiring & Onboarding

The selected candidate is hired and onboarded into the system. A new personnel number is created and the employee becomes active.

Key Information

T-Code: PA40 (Personnel Actions - Hire)
Action Type: 101 (Hire)

Hiring Steps

  • Issue Job Offer to selected candidate
  • Acceptance & Document Collection
  • Create PA40 Action (Hire)
  • Assign Personnel Number
  • Set Employee Status to Active
  • Onboarding & System Training

Onboarding Infotypes

Infotype Description
IT0001 Organizational Assignment
IT0002 Personal Data
IT0006 Addresses
IT0008 Basic Pay
IT0009 Bank Details

Employee Master Data Maintenance

Recording, updating, and managing all employee-related information throughout their entire lifecycle.

Key T-Codes

PA30: Maintain Master Data
PA20: Display Master Data
PA40: Personnel Actions

Movement Action Types

  • 101: Hire - Create new personnel
  • 103: Termination - Set to Inactive
  • 105: Promotion - Update position/pay
  • 107: Transfer - Change area/position
  • 122: Pay Change - Update salary

Master Data Objects

Master Data Purpose
Personnel Master Employee information (pay, org, benefits)
Organizational Structure Company org units & hierarchy
Jobs & Positions Role classification
Cost Centers Where costs are incurred
Time Management Work schedules & attendance

Payroll Processing & Benefits

Calculating employee salaries, allowances, deductions, and statutory contributions. This is the Three-Way Payroll Integration.

Key T-Codes

PE01: Payroll Driver
PT60: Time Evaluation
PC00_M99_CALC: Payroll Calculation

Three-Way Payroll Integration

  • 1st Point: Employee Master Data (PA30/PA40)
  • 2nd Point: Time Evaluation Results (PT60)
  • 3rd Point: Payroll Run Results (PE01)

Payroll Run Process

Step Action
1 Navigate to PE01 & enter payroll area/period
2 Select employee selection criteria
3 Run simulation payroll first
4 Verify gross pay, deductions, compliance
5 Execute live payroll run
6 Post results to FI accounting

Termination & Offboarding

Final step in the HR lifecycle. Completes the full & final settlement and ensures all dues, deductions, and statutory obligations are processed.

Key T-Codes

PA40: Personnel Actions (Termination)
F110: Automatic Payment Program
F-53: Manual Payment

Termination Steps

  • Initiate Termination Action in PA40
  • Update Infotypes (0000, 0001, 0003)
  • Record exit details & last working day
  • Stop future payroll processing
  • Trigger final settlement process
  • Process final payments

Final Settlement

Component Description
Final Salary Salary up to last working day
Leave Encashment Payment for unused leave
Gratuity Severance payment (if applicable)
Deductions Taxes, PF, insurance, loans
Net Payment Final amount to employee
